# 推荐项目骨架说明

## 目标

这份说明用于把主架构文档落到实际 Python 工程结构，便于直接开始编码。

## 一、推荐工程分层

### 1. `app/`

应用启动与依赖装配层。

职责：

1. 启动 Qt 应用
2. 初始化配置、日志、数据库
3. 注册 ASR、翻译、导出适配器
4. 装配任务执行器与用例服务

建议文件：

```text
app/
  main.py
  bootstrap.py
  container.py
```

### 2. `ui/`

桌面界面层。

职责：

1. 主窗口
2. 项目列表
3. 任务进度
4. 参数面板
5. 日志与导出记录

建议文件：

```text
ui/
  windows/
    main_window.py
    project_window.py
  dialogs/
    import_dialog.py
    export_dialog.py
    settings_dialog.py
  widgets/
    task_list.py
    progress_panel.py
    subtitle_preview.py
  viewmodels/
    project_vm.py
    task_vm.py
```

### 3. `core/`

业务核心层，不依赖 Qt、FFmpeg 具体实现。

职责：

1. 编排主流程
2. 定义实体和接口
3. 执行字幕生成、翻译、导出用例

建议文件：

```text
core/
  domain/
    entities.py
    enums.py
    value_objects.py
  dto/
    project_dto.py
    subtitle_dto.py
    task_dto.py
  ports/
    asr.py
    translator.py
    exporter.py
    repository.py
  usecases/
    create_project.py
    transcribe_video.py
    translate_subtitles.py
    export_video.py
  services/
    workflow_service.py
    subtitle_service.py
    checkpoint_service.py
```

### 4. `infrastructure/`

外部依赖实现层。

职责：

1. FFmpeg 调用
2. Whisper 调用
3. 云端 API 调用
4. SQLite 仓储
5. 本地任务调度

建议文件：

```text
infrastructure/
  media/
    ffmpeg.py
    ffprobe.py
    segmenter.py
  asr/
    faster_whisper_engine.py
  translation/
    openai_translator.py
    batch_builder.py
  subtitle/
    srt_writer.py
    ass_writer.py
    aligner.py
    formatter.py
  export/
    burn_in_exporter.py
    soft_subtitle_exporter.py
  storage/
    sqlite_db.py
    project_repository.py
    task_repository.py
    subtitle_repository.py
  task/
    job_queue.py
    worker.py
    progress_bus.py
  logging/
    setup.py
```

### 5. `config/`

配置层。

职责：

1. 默认配置
2. 用户配置
3. 模型参数
4. API 设置

建议文件：

```text
config/
  settings.py
  default.toml
  profiles/
    cpu.toml
    gpu.toml
```

### 6. `tests/`

测试层。

职责：

1. 单元测试
2. 集成测试
3. 小样本视频验证

建议文件：

```text
tests/
  unit/
  integration/
  fixtures/
```

## 二、建议优先实现的接口

### ASR 接口

```python
class AsrEngine:
    def transcribe(self, audio_path: str, language: str | None, use_vad: bool) -> list:
        ...
```

### 翻译接口

```python
class Translator:
    def translate_segments(self, segments: list, source_lang: str, target_lang: str) -> list:
        ...
```

### 导出接口

```python
class VideoExporter:
    def export(self, video_path: str, subtitle_path: str, output_path: str, mode: str) -> str:
        ...
```

## 三、MVP 首批用例

建议先落这 4 个用例：

1. `CreateProject`
2. `TranscribeVideo`
3. `TranslateSubtitles`
4. `ExportVideo`

这样能最快打通端到端主链路。

## 四、开发顺序建议

1. 先完成 `FFmpeg + faster-whisper + SRT 导出` 的命令行验证
2. 再封装成 `core + infrastructure`
3. 最后接入 `PySide6 UI`

这样可以先验证核心流程，再做桌面交互层，能降低早期返工。
